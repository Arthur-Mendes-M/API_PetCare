Sobre o PetCare.

PetCare é a forma simples, organizada e centralizada em um sistema multi-plataforma para te auxiliar na gestão do seu petshop.

Sobre a API

Este é o endpoint raiz da API do PetCare. Aqui você terá um guia completo de como utilizar todos os endpoints da API PetCare connect.

IMPORTANTE:

O deploy pela plataforma Render, no plano gratuito (utilizado para esta API), deixa o servidor inativo após 15 minutos sem requisições! Portanto, caso sua primeira requisição demore, é só aguardar, não demora muito.
  • Os caracteres "<" e ">" não devem ser incluídas nas URL's
  • Ao enviar requisições que possuam corpo (body) se atente a forma como as chaves dos objetos devem ser escritos

Token de segurança

Por motivos de segurança, foi criado um token secreto que deve ser incluído em TODOS os demais endpoints, como demonstrado na imagem.

O token é lido através do query parameter (parâmetro de busca) nomeado auth onde, na url do endpoint a ser utilizado, é necessário específica-lo. Como no exemplo: https://api-petcare.onrender.com/employees?auth=<TOKEN> Obs: os caracteres "<" e ">" devem ser removidos em todos as url's. Servem somente para dar enfase no que deve ser passado.

Esse token é único e exclusivamente gerado para o uso desta API de forma interna pelas plataformas do PetCare.

Portanto, essa API não é pública.
